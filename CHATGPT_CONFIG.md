# Configuração do ChatGPT Integration
# ===================================

# Para usar a integração com ChatGPT, você precisa:

## 1. Obter uma API Key da OpenAI
#    - Acesse: https://platform.openai.com/api-keys
#    - Crie uma conta ou faça login
#    - Gere uma nova API key

## 2. Configurar a API Key (escolha uma opção):

### Opção A: Variável de ambiente (recomendado)
# export OPENAI_API_KEY="sua_api_key_aqui"

### Opção B: Inserir diretamente na interface
# Cole sua API key no campo "🔑 API Key OpenAI" na aba "🤖 Insights ChatGPT"

## 3. Custos da API
# - GPT-3.5-turbo: ~$0.002 por 1K tokens
# - Cada análise usa aproximadamente 500-1500 tokens
# - Custo médio: $0.001-0.003 por insight

## 4. Recursos Disponíveis:
# ✅ Comentário de mercado contextual
# ✅ Análise profunda de matrizes V-Cov  
# ✅ Insights estratégicos sobre ponderação alfa
# ✅ Recomendações práticas para portfólios

## 5. Exemplo de uso:
# 1. Configure sua API key
# 2. Vá para a aba "🤖 Insights ChatGPT"
# 3. Escolha o tipo de análise
# 4. Insira os tickers desejados
# 5. Clique no botão correspondente
# 6. Aguarde a análise do ChatGPT!

## Segurança:
# ⚠️  Nunca commite sua API key no repositório
# ✅ Use variáveis de ambiente ou insira manualmente
# ✅ Mantenha sua API key privada e segura

## Suporte:
# Em caso de problemas, verifique:
# - Conectividade com internet
# - Validade da API key
# - Créditos disponíveis na conta OpenAI
